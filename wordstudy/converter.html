This utility will allow you to convert a dictionary in a - b format, to a valid WordStudy dictionary.

Every line should be one entry, in a - b format. For example "apple - jab≈Çko", without the quotes of course.

Then, click the Convert button below the textbox, and copy the generated WordStudy dictionary.

Once you do, head over to the main WordStudy page, and import the copied dictionary.

IMPORTANT: If you'd like to use the hyphen (-) and make the converter not treat it as if it was the

separator of the entry, put a backslash before the hyphen, like so: \-
