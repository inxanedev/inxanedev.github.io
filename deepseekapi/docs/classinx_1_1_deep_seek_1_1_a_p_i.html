static

Performs a single blocking completion request to DeepSeek without creating an instance of the API class.

This is a static method, so you don't have to create an instance to use this.

It creates its own message history containing only the system prompt and the user message.

Note that this function simply creates a temporary API instance internally, so this doesn't provide any performance benefits over creating an instance yourself.

This should only be used when you only want to perform a single request during the whole lifetime of your application.

Parameters
